# Project Spec: atpanyuniverse-web

*Last updated: 2026-03-13*

## Overview

Panyu’s personal portfolio site at atpanyuniverse.co. Single-page site with sections for about, skills, work history/projects, and contact, with smooth animations and responsive layout.

## Tech Stack

- Frontend: Static HTML/CSS/JS
- Styling: plain CSS
- Hosting: GitHub Pages (CNAME: atpanyuniverse.co)
- Tooling: npm, ESLint, Prettier

## Features

### Backlog

- Hero and navigation (about, skills, work, contact)
- About section with bio and photo
- Skills section with visual skill levels
- Work/portfolio section with categories (code vs others)
- Contact section (form or links)
- Animations for headings, skills, and scroll
- GitHub Pages deploy pipeline and CNAME support
